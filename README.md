# photosense
An open-source AI image editing tool built for Photoshop, supporting GPT Image 2 and Nano Banana APIs for mask-based generative editing, inpainting, and seamless non-destructive workflow integration.

一款面向 Photoshop 工作流开发的开源 AI 图像编辑工具，支持 GPT Image 2 与 Nano Banana API，可实现基于蒙版的生成式重绘、局部修复以及无损的 Photoshop 工作流集成。
